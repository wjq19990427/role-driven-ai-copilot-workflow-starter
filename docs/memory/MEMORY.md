# 会话内存索引

> 跨会话持久记忆。每次对话开始时读此索引，按需读具体记忆文件。
> 只有 `/review` 角色能写记忆。

## 索引规则

- 每行 ≤ 150 字
- 格式：`- [标题](文件名) — 一句话描述`
- 项目刚启动时此处为空，由 `/review` 角色在工作中逐步沉淀
